# Modelowanie-i-Predykcje---Berka-Dataset
Stworzenie modelu predykcyjnego, który przewidzi, czy klient nie spłaci pożyczki. Analizie zostały poddane dane pochodzące z Berka Dataset (source: https://www.kaggle.com/datasets/marceloventura/the-berka-dataset?resource=download)

Projekt składa się z następujących części: przygotowania danych, eksploracyjnej analizy danych (EDA), budowy modelu predykcyjnego, przetestowania różnych algorytmów (m.in. regresji logistycznej, drzew decyzyjnych, lasów losowych, XGBoost) oraz porównania ich wyników na podstawie metryk (m.in. dokładności, precyzji, recall, F1-score) i wyciągnięcia adekwatnych wniosków.
